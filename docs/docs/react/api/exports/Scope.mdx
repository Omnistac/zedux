```ts
import { Scope } from '@zedux/react'
```

An enum of the possible atom scopes. Scope determines many characteristics of an atom, primarily:

- How instances are created
- How long instances live
- What hooks/injectors are available
- Where it can be used (e.g. app atoms can't be injected into global atoms)

## Definition

```ts
enum Scope {
  App = 'App',
  Global = 'Global',
  Local = 'Local',
}
```

### `App`

Used to create/identify [`AppAtom`](../types/AppAtom)s.

<LiveEditor resultVar="App">{`
const appAtom = atom({
  key: 'app',
  scope: Scope.App,
  value: 'app val!'
})\n
function Child() {
  const [app, setGlobal] = appAtom.useState()\n
  return (
    <input
      onChange={({ target }) => setGlobal(target.value)}
      value={app}
    />
  )
}\n
function App() {
  return (
    <>
      <Child />
      <AppProvider>
        <Child />
        <Child />
      </AppProvider>
    </>
  )
}
`}</LiveEditor>
