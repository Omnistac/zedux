An AtomConfig object is the 3rd argument passed to [the `atom()` factory](../factories/atom).

## Definition

```ts
interface AtomConfig {
  flags?: string[]
  maxInstances?: number
  ttl?: number
}
```

### `flags`

Optional. An array of strings. These flags correlate to the [Ecosystem's `flags` property](../classes/Ecosystem#flags). See that page for an example.

### `forwardPromises`

Optional. A boolean. Whether instances of this atom should create a default [`.promise` property](../classes/AtomInstance#promise) that is a `Promise.all()` of the atom instance's initial dependencies' promises. See [Atom#forwardPromises](../classes/Atom#forwardpromises).

### `maxInstances`

Optional. A number. Indicates the largest ideal cache size for this atom. See [Atom#maxInstances](../classes/Atom#maxinstances).

### `ttl`

Optional. A number. How long instances of the atom should stick around when stale. See [Atom#ttl](../classes/Atom#ttl).
