import { LiveEditor } from '../../../LiveEditor'

```ts
import { injectEcosystem } from '@zedux/react'
```

An [injector](../glossary#injector) that returns a reference to the [ecosystem](../classes/Ecosystem) the current atom instance was (or is being) created in.

Can be used to inspect other atom values without registering a graph dependency on those atom instances.

:::important
This should be viewed as an escape hatch. There shouldn't be many use cases for accessing the ecosystem inside an evaluator function. Use sparingly.
:::

## Example

<LiveEditor resultVar="Seconds">{`
const secondsAtom = atom('seconds', () => {
  const store = injectStore(0)\n
  injectEffect(() => {
    const intervalId = setInterval(
      () => store.setState(val => val + 1),
      1000
    )\n
    return () => clearInterval(intervalId)
  }, [])\n
  return store
})\n
const lazyAtom = ion('lazy', ({ get }) => {
  const currentEcosystem = injectEcosystem()\n
  // ion's get function would register a dynamic graph dependency here.
  // Try deleting "currentEcosystem.":
  const seconds = currentEcosystem.get(secondsAtom)\n
  return seconds
})\n
function Seconds() {
  const lazySeconds = useAtomValue(lazyAtom)
  const instance = useAtomInstance(lazyAtom)\n
  return (
    <>
      <div>Unchanging Seconds: {lazySeconds}</div>
      <button onClick={() => instance.invalidate()}>Force Update</button>
    </>
  )
}
`}</LiveEditor>

## Signature

```ts
injectEcosystem() => Ecosystem
```

Returns [an Ecosystem instance](../classes/Ecosystem).
