```ts
import { ecosystem } from '@zedux/react'
```

Creates an [atom ecosystem](../classes/Ecosystem).

## Example

```ts
import { ecosytem } from '@zedux/react'

const rootEcosystem = ecosystem({ id: 'root' })

const withOverrides = ecosystem({
  id: 'withOverrides',
  overrides: [atomA, atomB]
})

const withPreloadFn = ecosystem({
  id: 'withPreloadFn',
  preload: es => { // `preload` is passed a reference to the ecosystem
    es.getInstance(myAtom) // preload this atom instance
  }
})
```

## Signature

```ts
ecosystem(config?) => Ecosystem
```

### `config`

Optional. An [EcosystemConfig object](../types/EcosystemConfig).

All config fields are optional. But it is recommended to at least pass an id.
