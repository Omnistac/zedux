### `.invalidate()`

All standard, query, and mutation atom instances have an `invalidate` function to force reevaluation:

<LiveEditor resultVar="Coin">{`
const coinTossAtom = atom('coinToss', () => Math.random() < 0.5)\n
function Coin() {
  const isHeads = coinTossAtom.useValue()
  const { invalidate } = coinTossAtom.useInstance()\n
  return <button onClick={invalidate}>{isHeads ? 'Heads' : 'Tails'}</button>
}
`}</LiveEditor>
